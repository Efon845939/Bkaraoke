# **App Name**: Karaoke Queue

## Core Features:

- Song Submission: Students can submit song requests, including song title and a link to a karaoke video.
- User Authentication: Secure user authentication to differentiate between students and admins.
- Song List Display: Display a searchable, sortable list of requested songs, viewable by all users.
- Admin Panel: Dedicated admin panel to view song requests with student information and manage the list (edit, delete).
- User Attribution: Record and display the student's name alongside their submitted song request for admins.

## Style Guidelines:

- Primary color: Black (#000000) for a classic 90s feel.
- Background color: Light gray (#D3D3D3) to complement the black and white theme.
- Accent color: Dark gray (#A9A9A9) to highlight interactive elements and call-to-actions.
- Headline font: 'Arial Black' sans-serif font, reminiscent of 90s typography.
- Body font: 'Helvetica' a clean, readable sans-serif font.
- Use simple, geometric icons inspired by 90s design (e.g., pixelated images, cassette tapes).
- Subtle animations, such as fading or sliding effects, with a slightly retro or glitchy feel.
